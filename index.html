Elevo
AI Placement Prep Platform · Full-Stack Web App
Live Demo GitHub
Overview
Elevo is an AI-powered placement preparation platform built to bridge the gap between academic learning and industry readiness. It combines coding practice, aptitude prep, resume-based mock interviews, tutor-led learning, analytics, and performance tracking into one structured system for both individual learners and organizations such as colleges or training institutes.
Core Features
  • Unified preparation workflow — coding practice, aptitude tests, resume analysis, AI mock interviews, and readiness tracking in one platform instead of multiple disconnected tools.
  • Role-based access control — supports students, tutors, admins, organization members, and institutional use cases with separate permissions and dashboards.
  • Resume-based interview engine — analyzes uploaded resumes and generates tailored technical or HR interviews with AI feedback.
  • Social learning and tutor content — includes a learning feed, tutor-managed material, and guided preparation resources.
  • Analytics + subscription features — tracks performance trends, learner progress, and gated premium access for scalable rollout.
  • Live coding practice — supports remote code execution and test-case validation for programming rounds.
Hardest Challenge
🔥 Platform orchestration — The main challenge was designing one cohesive system where AI interviews, resume parsing, role-based flows, coding execution, analytics, and subscription logic all work together cleanly inside the same product without feeling fragmented.

Bloom Wellness
Mental Wellness Platform · Full-Stack Django App
Live Demo GitHub
Overview
Bloom is a full-stack mental wellness web application built with Django that combines emotional self-care tools with therapist discovery and appointment booking. It brings journaling, mood tracking, wellness chat, insights, and access to professional care into one connected digital mental health ecosystem.
Core Features
  • User wellness workspace — account management, personal profiles, mood logging, reflective journal entries, and recent emotional history.
  • Therapist discovery + booking — searchable therapist profiles, availability management, appointment booking, rescheduling, cancellation, reviews, and patient messaging.
  • Hybrid AI wellness layer — chatbot sessions with sentiment and emotion analysis, contextual response logic, crisis keyword handling, coping strategies, and wellness goals.
  • Insights dashboard — combines journal, mood, and chatbot activity into charts, KPIs, summaries, and pattern-based wellness insights.
Hardest Challenge
🔥 Balancing care workflows with product depth — The challenge was designing a platform that supports both patient self-reflection and therapist operations without turning into a disconnected feature set. The chatbot and therapist utilities needed to feel supportive, context-aware, and responsible while still fitting a server-rendered Django architecture.
